# API Documentation

## Функции калькулятора

### add(a, b)
Складывает два числа.
- **Параметры:** a (float), b (float)
- **Возвращает:** float

### subtract(a, b)
Вычитает второе число из первого.
- **Параметры:** a (float), b (float)
- **Возвращает:** float

### multiply(a, b)
Умножает два числа.
- **Параметры:** a (float), b (float)
- **Возвращает:** float

### divide(a, b)
Делит первое число на второе.
- **Параметры:** a (float), b (float)
- **Возвращает:** float или str
- **Особенности:** Проверка деления на ноль

### power(a, b)
Возводит число в степень.
- **Параметры:** a (float), b (float)
- **Возвращает:** float

### mod(a, b)
Возвращает остаток от деления.
- **Параметры:** a (float), b (float)
- **Возвращает:** float или str
